# Project Overview: Fortex Landing Page

## Description
This project is a high-performance, conversion-oriented landing page for **Fortex Digital Solutions**. The goal is to present Fortex as a premium B2B partner for operational digital transformation. The design focuses on "Order vs. Chaos," showing how Fortex brings control and visibility to complex operations.

## Tech Stack
- **Framework:** Astro (v5.0+)
- **UI Libraries:** React (for interactive components)
- **Styling:** Tailwind CSS (v4.0) with custom configuration
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **Utilities:** `clsx`, `tailwind-merge`

## Key Value Propositions
1.  **Transform Chaos into Order:** The core narrative.
2.  **Suite OneTex:** The flagship product for operational management.
3.  **Process-First Approach:** Analysis before code.

## Directory Structure
- `src/components`: React and Astro components.
- `src/layouts`: Main layout files (Layout.astro).
- `src/pages`: Astro pages (index.astro, demo.astro).
- `src/styles`: Global CSS and Tailwind directives.

## Recent Updates
- **Theme Change:** Switched from Orange accent to **Blue** (`blue-500`, `blue-600`) to align with professional B2B aesthetics.
- **New Page:** Added `/demo` page with a contact form.
- **Enhanced Hero:** Larger typography, softer backgrounds, improved dashboard visualization.
